# Arquitetura_monolitica_vs_microsservi-os
Arquitetura monolítica: todo o sistema é construído como um único bloco. Mais simples de desenvolver no início, mas difícil de escalar e manter conforme cresce. Microsserviços: o sistema é dividido em vários serviços pequenos e independentes. Mais flexível e escalável, porém mais complexo de gerenciar.
